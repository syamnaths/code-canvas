How to Use

Welcome to CodeCanvas! Using the platform is straightforward:

  1. Choose Your Course: Use the dropdown in the sidebar to switch between different learning paths (e.g., HTML/CSS or JavaScript).
  2. Select a Lesson: Choose a topic from the navigation sidebar.
  3. Read and Learn: Study the content provided in the main area.
  4. Pass the Quiz: Each lesson has a topic quiz. You must score at least 70% to unlock the interactive playground.
  5. Practice Your Code: Once unlocked, use the editor to experiment. Your results appear instantly in the live preview.

Key Features

  • Smart Live Preview: Our preview engine intelligently handles both code snippets and full HTML documents.
  • Progress Tracking: Your quiz results and capstone project code are automatically saved to your browser's local storage.
  • Curriculum Switching: Easily toggle between multiple courses without losing your place.
  • Data Portability: Use the Preferences page to export your entire progress as a JSON file and restore it on another device.

Frequently Asked Questions

Why is the Practice Ground locked?

To ensure you understand the core concepts, we require a passing score (70%+) on the topic quiz before you can start coding.

Is my code saved?

Yes! Your progress and capstone project code are saved locally. However, clearing your browser data will erase this. Use the Export JSON feature on the Preferences page for long-term backups.
