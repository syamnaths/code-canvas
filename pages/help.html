How to Use

Welcome to CodeCanvas! Using the platform is straightforward:

  1. Select a Lesson: Choose a topic from the navigation sidebar on the left.
  2. Read the Content: The lesson's explanation and instructions will appear in the main content area.
  3. Practice Your Code: Use the "HTML & CSS Editor" to write your code. The initial code for the lesson is loaded for you automatically.
  4. See Instant Results: Your code's output will render instantly in the "Live Preview" panel on the right.

Features

  • Interactive Editor: A split-screen view allows you to code and see your changes in real-time without leaving the page.
  • Dynamic Lessons: All lessons are loaded from a central lessons.json file, making the course content easy to update and expand.
  • Modern UI: Built with Tailwind CSS for a clean, responsive, and modern user interface that works on any device.

Frequently Asked Questions

Can I add my own lessons?

Yes! You can edit the assets/data/lessons.json file directly. Simply copy the format of an existing lesson (title, content, initialCode) to add your own content.

Is my code saved?

Your code in the editor is not saved automatically between sessions in this version. Use the features on the Preferences page to export your lesson file if you want to save your progress.
